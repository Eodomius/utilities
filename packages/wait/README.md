# Eodomius utilities : cast

Utils packages for Eodomius project

## Installation

**NPM**:

```bash
npm i @eodomius/wait
```

**Yarn**:

```bash
yarn add @eodomius/wait
```

## Importation

**CommonJS**:

```js
const { wait } = require("@eodomius/wait");
```

**EcmaScript modules**:

```js
import { wait } from "@eodomius/wait";
```

## Usage

```typescript
import { wait } from "@eodomius/wait";

await wait(1000);
```
